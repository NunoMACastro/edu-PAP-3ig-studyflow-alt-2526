# HTML e CSS - Capítulos (11.º ano)

Coleção de 16 capítulos para consolidar bases de frontend e preparar transição natural para DOM em JavaScript e componentes em React.

## Como usar

1. Segue a ordem `01` -> `16`.
2. Testa todos os exemplos num projeto com esta estrutura:

```text
mini-projeto/
  index.html
  css/
    base.css
    layout.css
    components.css
  js/
    main.js
```

3. Em cada capítulo: faz o Exemplo A, identifica a armadilha do Exemplo B e resolve os exercícios A/B/C.

## Convenções da coleção

- Classes base: `.cartao`, `.botao`, `.mensagem`, `.input`, `.container`, `.barra`, `.grelha`.
- Estados: `.is-loading`, `.is-error`, `.is-success`, `.is-empty`, `.hidden`, `.is-done`.
- `id` para seleção no DOM e `class` para estilo/estado.
- PT-PT consistente em títulos, exemplos e exercícios.
